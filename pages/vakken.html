Sprint 0

Dit was het eerste vak waar we mee bezig gingen tijdens de minor. Bij dit project kwam je er achter wat je allemaal al kon. Eigenlijk een soort 0 meting. Ik heb heirbij eigenlijk gewoon een website gemaakt met alles dat ik al kon of wat ik wist dat bestaat en dat opgezocht hoe dat moet. Ik heb hierbij ook persoonlijke dingen verwerkt, zoals het stukje met de auto. Dat is een roadtrip met alle landen waardoor ik een roadtrip maakte.

BrowserTech

Bij dit vak moesten we een erfbelastingformulier omzetten naar digitaal. Maar dan in de stijl van de ns. Ik heb hier veel bij geleerd, zoals progressive disclosure en hoe het zit met het fact checken van data, zoals de elf-proef. Dit was best een leuk project dat er van de buitenkant niet heel ingewikkeld uit ziet, maar wel heel veel achter zit. Ook heb ik met een datalist gewerkt en als je het land opzoekt geeft die je automatisch de landcode

CSS To The Rescue

Bij dit project heb ik geleerd hoe je alles moet doen met css. Voor veel verschillende functies wil je heel snel javascript gebruiken, terwijl css tegenwoordig heel krachtig is en heel veel van die dingen al kan. Een van de leukste onderdelen waar ik ook het meeste plezier mee had met uittesten was het werken met style containers. Je kan zo heel makkelijk met een knop alles in een keer laten veranderen. Dit vind ik ook oprecht een best hadige container functie. Voor de rest was dit opzich best een leuk vak.

API

Bij dit vak heb ik meer geleerd dan ik dacht. Het ging mij dan niet perse hoe je een api inlaad, maar meer over hoeveel er eigenlijk wel niet beschikbaar zijn. Web api's vond ik vooral heel handig. Het maakt een website veel gaver, en zit vaak al ingebouwd in je browser. Je hoeft het alleen nog maar aan te roepen. Ik vond hier specifiek de viewTransition api heel cool. Heel makkelijk en ziet er heel gaaf uit.
